工場生産ライン監視システム

プロジェクト概要：

本システムは、屋根材製造会社向けに開発された生産ライン監視システムであり、 各顧客注文のリアルタイムな生産状況を追跡・管理することを目的としています。 このWebベースのソリューションは、 工場の品質管理チームと本社営業部門をつなぎ、 両者が生産の進捗を即座に確認できるようにします。 リアルタイムのデータ更新により、生産部門と営業部門の連携を強化し、 手作業による報告を最小限に抑えることで業務効率を向上させました。 システムには、注文状況、生産スピード、完了率をリアルタイムで表示する 中央ダッシュボードが備わっており、 管理者はデータに基づいた迅速な意思決定が可能となります。

主な機能：

  • リアルタイムでの注文追跡とステータス更新
  • 品質管理担当者による工場側データ入力と検証
  • 本社用ダッシュボードによる進捗の可視化
  • 自動レポート生成とパフォーマンス分析
  • セキュアかつスケーラブルなAPI駆動型アーキテクチャ

実績・技術構成：

  • フロントエンド： React.js
  • バックエンド： Node.js（Express）
  • 認証方式： JWT（JSON Web Token）
  • データベース： MySQL
  • リアルタイム通信： WebSocket
  • ホスティング・デプロイ： Linuxサーバー（Apache / Nginx）
経費・取引管理システム

プロジェクト概要：

本システムは、企業内の経費および取引を一元管理するために開発されました。すべての取引は、 財務部門や管理部門による承認フローを経て処理される仕組みを備えており、複数のユーザーレベルと承認段階に対応することで、 承認プロセスの自動化と取引の透明性向上を実現しています。本システムにより、経費管理の効率化、手作業によるミスの削減、 迅速な意思決定をサポートします。

主な機能：

  • 取引および経費データの登録・管理
  • ユーザーレベルごとの承認フロー自動化
  • 財務部門および管理部門による承認履歴の追跡
  • ダッシュボードによる取引状況の可視化
  • システム管理者向けアクセス制御とログ管理

実績・技術構成：

  • フロントエンド： React.js
  • バックエンド： Lumen（REST API）
  • 認証方式： OAuth
  • データベース： MySQL
  • ホスティング・デプロイ： Apache Web Server
  • チームリーダーとしてフルスタック開発、デプロイ、ユーザー研修を担当
店舗生産・在庫計画システム

プロジェクト概要：

本システムは、地元スーパーマーケットチェーン向けに開発され、店舗での生産状況を記録し、 売上・生産・残在庫のデータを基に翌日の生産計画を予測することを目的としています。これにより、食品廃棄の削減と効率的な生産管理を実現します。 システムは LAN 経由で本社サーバーと各店舗間のデータ通信を行い、リアルタイムでの情報共有を可能にします。

主な機能：

  • 店舗での生産記録および在庫状況の管理
  • 前日データを基にした翌日の生産計画予測
  • 本社サーバーと店舗間のリアルタイムデータ同期
  • POS システムからの売上データ取り込み（FTP通信）
  • デバイスおよびシステムのアクセス管理と運用サポート

実績・技術構成：

  • フロントエンド・バックエンド： PHP（CodeIgniter）、JavaScript、jQuery、HTML/CSS
  • データベース： MySQL
  • ネットワーク： LAN 通信
  • デプロイ・運用： Windows サーバー上でのシステム構築、店舗デバイス設定
  • 担当： 要件定義、システム・データベース・UI設計、フルスタック開発、統合・単体テスト、システムデプロイ
目で操作する音楽プレイヤーアプリケーション

プロジェクト概要：

愛媛県の共同プロジェクト「愛顔eスポーツ」の一環として、モビリティに制限のあるユーザーが目の動きで音楽再生を操作できるカスタムアプリケーションを開発しました。 本アプリケーションは、ユーザーの能力に合わせて操作可能で、ミニPC、モニター、Bluetoothスピーカー、Omron Vision Component を統合したシステムにより、アクセシブルで参加型のeスポーツ体験を提供します。

主な機能：

  • 目の動きによる音楽再生・停止・曲送り操作
  • ユーザーの能力に応じた操作設定のカスタマイズ
  • ハードウェア（ミニPC、モニター、スピーカー、センサー）との統合
  • ユーザー向け操作ガイド作成と使用サポート

実績・技術構成：

  • 開発環境： .NET Windows Forms
  • デバイス連携： Omron Vision Component による目の動き検知
  • システムテストおよび納品プロセスへの参加
  • フルスタック開発、ユーザーガイド作成、セットアップ環境構築を担当
